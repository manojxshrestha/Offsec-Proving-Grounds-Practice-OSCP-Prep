## How to Use This Repository

1. **Browse by Lab**: Navigate to any lab directory to find its specific documentation
2. **Follow the Process**: Each README demonstrates the penetration testing methodology:
   - Initial reconnaissance and enumeration
   - Vulnerability identification
   - Exploitation techniques
   - Post-exploitation and privilege escalation
3. **Learn from Examples**: 
   - Study the commands and tools used
   - Understand the reasoning behind each step
   - Reference the screenshots for visual confirmation of expected outputs
4. **Apply to Your Practice**: Use these notes as a reference when attempting similar machines in the Offsec Proving Grounds lab environment
